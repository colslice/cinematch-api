# Cinematch

Backend REST API for the Cinematch MERN application (Node.js, Express, MongoDB).

## Setup

**Install Dependencies** Run this in the root folder:
```bash
npm install
```

## Environment Variables

Create a file named `.env` in the root folder of this project. These will be the contents of the file:

```env
PORT=5001
MONGO_URI=your_database_connection_string_here
```

## Running the Server

Start the server by running:
```bash
npm run dev
```
